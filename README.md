# social-pulse-
SocialPulse is a social media management dashboard for agencies — think of it as a mini Hootsuite/Buffer, built for someone managing multiple clients' Instagram, Twitter/X, and LinkedIn accounts from one place

Social Pulse

✨ Overview
A polished Android app with a premium SaaS look — dark theme, green/purple gradient accents, glassy cards, and smooth animations.
Currently UI/UX is 100% complete across 7 screens with realistic demo data. Backend integration is the next milestone.

🚀 Walkthrough
Splash → Animated logo bounce-in, tagline “Manage. Schedule. Grow.”

Login → Email/password + Google Sign-In, gradient button with loading state

Home Dashboard → Client switcher, stat cards (Reach, Engagement, Posts Today, New Followers), weekly bar chart, upcoming posts

Schedule → Calendar week view, post status badges (Published/Scheduled/Draft), swipe actions (edit/duplicate/delete)

Compose → Multi-platform post composer, caption counter, attach media, schedule/publish

Analytics → Date-range selector, impressions chart, engagement metrics, top posts, best-time-to-post insight

Settings → Profile, connected accounts, notification prefs, dark mode, plan upgrade banner, log out

Navigation: bottom nav bar (Home / Schedule / [+] / Analytics / Settings).

🔜 Dynamic Backend (Coming Soon)
Real stats pulled from Firestore + social APIs

OAuth connections to Instagram, Twitter, LinkedIn

Compose → publish/schedule posts directly to platforms

Schedule/Analytics → reflect real post history

Multi-client switching → live client list from Firestore

✅ Current Feature Set
Multi-platform post composer (Instagram, Twitter, LinkedIn)

Scheduling with calendar view + status tracking

Analytics dashboard with charts & top-post ranking

Multi-client dashboard structure

Firebase email/Google auth (auth is real; data is demo)

Dark, branded UI system

Settings: notification prefs, account connections UI, plan/upgrade banner
